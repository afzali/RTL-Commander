Privacy Policy

Last updated: June 2024

Introduction

RTL Commander respects your privacy and is committed to protecting any data you provide while using this extension. This Privacy Policy explains what data we collect, how we use it, and your rights regarding that data.

1. Information We Collect

This extension does not collect, store, or share any personal data. However, to improve user experience, the following may be stored locally on your device:

  • User Preferences: Text alignment settings (RTL/LTR) configured by the user.
  • Context Menu Actions: User selections for applying alignment settings.

No data is transmitted to external servers or third parties.

2. How We Use the Data

The locally stored data is used only to:

  • Allow users to adjust text direction dynamically.
  • Remember user preferences for a consistent browsing experience.

3. Permissions We Require

RTL Commander requires the following Chrome permissions for proper functionality:

  • activeTab: To modify text direction on the active webpage upon user request.
  • contextMenus: To enable text alignment options in the right-click menu.
  • scripting: To inject scripts that apply the user's text direction settings.
  • storage: To save user preferences locally (not on external servers).
  • host permissions: To modify webpage content when necessary.

These permissions are used exclusively for providing the extension's intended features and not for collecting user data.

4. Data Security

Your preferences are stored only on your device. RTL Commander does not send, track, or store any personal data externally.

5. Third-Party Sharing

We do not share, sell, or transmit any user data to third parties.

6. Changes to This Policy

Any updates to this Privacy Policy will be reflected on this page. We encourage users to check for updates periodically.
